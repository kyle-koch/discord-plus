What is this?

Discord+Sath is a theme for Discord. It's both sleek and customizable, without getting in your way. Its top priority is balancing between looks and performance. I have shamelessly stolen this from PlusInsta, yarr.

All you need to install my theme is Vencord. If you already have it, you're ready to install my theme!

User customization

This theme is very customizable. The variables in the theme file allow you to change many things, including but not limited to:

  • Background image
  • Accent color
  • Up to three fonts
  • Home icon
  • Background cover colors
  • UI corner roundness
  • Overall spacing

If customizing the theme is difficult, bitch at me and I'll help you.

Installation

Click Me This opens the GitHub page you need in a new tab.
Click the crumbs menu

Click Download

Paste %USERPROFILE%\AppData\Roaming\Vencord\themes into Windows File Explorer to go to your Vencord themes folder

Move the .css file you downloaded from me to the themes folder.

After that, you'll be good to go! It'll be available in your Vencord themes.
